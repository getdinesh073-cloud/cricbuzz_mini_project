# Cricbuzz API Reference

## API Endpoints

- Live Matches: Fetch ongoing cricket matches
- Match Scorecard: Get detailed match scorecard
- Player Stats: Retrieve player statistics
- Series Info: Fetch series and tournament information

## Authentication

- Use API key for authenticated requests
- Set CRICBUZZ_API_KEY in environment variables

## Error Handling

- Implement retry logic for failed requests
- Add timeout handling for API calls
- Log all API errors for debugging
